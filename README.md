# 叮点跑腿小程序 v2

## 项目介绍

本项目后端采用 midway3.0，后台采用 nuxt2.x,小程序采用 uniapp 实现的一套跑腿下单接单系统。

#### 主要功能

帮送服务、帮买服务、骑手注册、骑手接单、用户下单、提现、订单分配系统、优惠券、物品重量计算、距离计算等。
